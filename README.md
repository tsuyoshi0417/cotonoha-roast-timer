# 言ノ葉 焙煎タイマー v2.0
家庭焙煎の記録と再現のための無料ツール。ブラウザだけで動く（データは端末内 localStorage・Googleログインで任意のクラウド同期）。

- 焼く：タイマー＋節目ワンタップ（投入/中点/1ハゼ/2ハゼ/煎り止め）・温度カーブ・火力ログ
- 測る：重量減%・DTR・区間ROR（経験則の指標）・TDS・抽出収率EY・Brix（糖度の近似）
- 味わう：カッピング6軸を焙煎ロットに紐付け
- 次を決める：ロット一覧・A×B比較・AI解析用エクスポート（readme_for_ai同梱）

構成：`index.html`＋`roast-metrics.js`（計算）＋`roast-knowledge-data.js`（知識カードの差し込み口）。
